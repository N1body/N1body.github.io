Mysql命令笔记
2020-12-27

创建数据库

create database [数据库名];

设置默认数据库

use [数据库名];

创建表格

create table [表格名] ([列名] [类型] [null/not null] [默认值] [auto_increament] [unique key/primary key] [comment ‘列注释’] [参照定义]);

复制表格

create table [创建的新表格名] like [被复制的表格名]; 仅复制结构

*create table [创建的新表格名] as (select * from [被复制的表格名]);* 仅复制数据以及数据对应的部分结构

展示表格、数据库、数据

show databases; 查看数据库

show tables(设置了默认数据库的前提下); 查看表格

describe [表格名]; 查看具体表格结构

*select * from [表格名];* 查看表内所有数据

修改表格

alter table [表格名]

​ rename to [表格新名字]; 修改表格名

​ add column [列名] [类型] [null/not null] after [在某个列名之后]…; 增加列

​ drop column [列名]； 删除列

​ change [原列名] [新列名] [修改后的类型]; 修改列名及类型

​ modify [列名] [修改后的类型] [null/not null]; 修改类型

​ drop primary key; 删除主键

​ add primary key([列名]); 添加主键

​ drop foreign key; 删除约束

​ add constraint foreign key (sno) references student (sno); 增加约束

​

开启、关闭外键约束

set foreign_key_checks = 0; 关闭外键约束

set foreign_key_checks = 1; 开启外键约束

添加数据

insert into [表名] values ([对应类型的数据],….),(),…;

insert into [表名] set 列名=[数据],…..;

修改数据

update [表名] set [列名1] = 表达式 where 条件;

删除数据

delete from [表名] where 条件;

索引

添加索引 create [unique/fulltext] (唯一索引/全文索引) index [索引名] on [表名] ([列名1] [ASC/DESC],…)

删除索引 drop index [索引名] on [表名]

查询数据

select [列名],(表达式或函数 (重命名))… from [表名] where [条件]

条件可用特殊字符指定范围:

**where [列名] between [左边数字] and [右边数字] **

where [列名] in/not in ([集合]) //集合必须满足类型与前面条件限制的类型保持一致

where [列名] like ‘[字符串]’ //字符串可用通配符控制 %代表剩余的字符 _代表一个字符

注意： 如果要查询的字符串本身含有通配符 可在语句后加上 escape [‘通配符’]

空值查询用 is null/is not null 条件查询 不能用=

where [列名] [条件1] and/or [条件2]

升序排列： order by [列名] ASC //默认升序

降序排列： order by [列名] DESC

*select * from table_constraints where contraints_schema = ‘[数据库名]’ and table_name = [表名];* 查询约束

聚集函数

*count () //统计元组个数**

count( [distinct/all] (distinct清除重复值 默认all不清楚) [列名]) //统计指定列中的个数

sum( [distinct/all] (distinct清除重复值 默认all不清楚) [列名]) //计算指定列中的总和

avg( [distinct/all] (distinct清除重复值 默认all不清楚) [列名]) //计算指定列中的平均值

max( [distinct/all] (distinct清除重复值 默认all不清楚) [列名]) //计算指定列中的最大值

min( [distinct/all] (distinct清除重复值 默认all不清楚) [列名]) //计算指定列中的最小值

嵌套查询

select [列名] from [表名] where [条件]

条件： [表间有联系的列名] = (select [表间有联系的列名] from [有联系的表名] where [对应表内的条件])

连接查询

select [列名] from [表名1],[表名2]….(所有在连接中的表) where [表名1].[列名] = [表名2].[列名] and 条件 ……

触发器

 [触发内容]

delimiter ;

存储过程（类似函数,只不过返回值也在参数里）

create procedure [存储器名] (参数(不能默认值，无法省略则用null代替))

begin

​ 存储体(每条语句都要加上;)

end

**参数：in/out/inout [参数名] [参数类型] **

存储体：定义变量：declare [变量名] [变量类型];

​ 改变变量值: set [变量名] = [表达式]

设置临时变量: set @[临时变量名] = [值]

调用存储体: call [存储器名] (参数(in)/@参数(out/inout))

**语句块命名: **

[语句块名]:begin

​ [语句块]

​ leave [语句块名] (跳出语句块)

end [语句块名]

展示信息: show procedure status (显示所有存储过程信息) show create procedure [存储器名] 显示某一个存储过程的详细详细信息

流程控制语句

if语句:

if [条件] then

​ 语句;

elseif [条件] then

​ 语句;

**else **

​ 语句;

end if;

while语句:

[标签名]: while [条件] do

​ 语句;

​ leave [标签名] (跳出循环)

end while [标签名]

**case语句: **

case 表达式

​ when [值1] then 语句2;

loop语句:

[标签名]:loop

​ leave [标签名]

end loop [标签名]

游标

使用游标的五个步骤: 定义游标 打开游标 使用游标 关闭游标 释放游标

**定义游标: declare [游标名] cursor for [select语句] **

打开游标: open [游标名]

使用游标: fetch [游标名] into [变量名],[变量名]… (子句中变量的数目必须等于游标一行数据中列的数目)

​ fetch [next | prior | first | last] from [游标名] [ into 变量名1,变量名2,变量名3[,…] ]

​ next：取下一行的数据，游标一开始默认的第一行之前，故要让游标指向第一行，就必须第一次就执行FETCH NEXT操作

​ prior: 取上一行的数据

​ first: 取第一行的数据

​ last: 取最后一行的数据

​ into：将一行中每个对应的列下的数据放到与列 的数据类型相同的变量中。

关闭游标: close [游标名]

declare continue handler for NOT FOUND [set语句(用来判断游标是否越界)];

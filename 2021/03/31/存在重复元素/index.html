题解：存在重复元素
2021-03-31|刷题
字数总计: 487|阅读时长: 2 分钟

题目介绍

给定一个整数数组，判断是否存在重复元素。

如果任意一值在数组中出现至少两次，函数返回 true 。如果数组中每个元素都不相同，则返回 false 。

输入: [1,2,3,1]
输出: true

输入: [1,2,3,4]
输出: false

输入: [1,1,1,3,3,4,3,2,4,2]
输出: true

1. 暴力双重循环【数组】

暴力双重循环，时间复杂度O(n^2)，当数据超过10^5时，就会超时.

2. 先排序，后查找【排序】

用时间复杂度低的排序算法先对数组进行排序（例如工具类Arrays.sort()，时间复杂度n*logn，原理自行百度），之后查找log(n)，最终取决于排序的时间复杂度。缺点：会对原数组更改，所以做题先最好先复制原数组.

3. 哈希表查找【哈希表】

哈希表查找的时间复杂度是O(1)，将数组中的元素全部放入哈希表中，并且在每次放入之前查找哈希表中是否有重复元素。

4. 二叉树查找【二叉树】

二叉树查找的时间复杂度是O(logn)，将数组中的元素全部放入哈希表中，并且在每次放入之前查找哈希表中是否有重复元素。
